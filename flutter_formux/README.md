A range of tools to simplify reactive form validation in Flutter.

## Features

- ✅ Form validations
- 🧩 BLoC focus
- 🌎 Fetching data
- 🪄 Error validation messages
- 📦 Components available

## Usage

Add the library in your project

```console
$ flutter pub add formux
```
